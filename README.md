Program that facilitates learning foreign language vocabulary. Features:
1. Displaying a list of dictionary files
2. Selecting a dictionary file
3. Selecting a language from the learning file
4. Entering words and checking their correctness
5. Entering a word incorrectly means that you have to enter it correctly n+1 times, where n is the number of unsuccessful attempts made so far
6. Correct completion of the word list completes the program.
